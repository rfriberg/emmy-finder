# Emmy finder
Simple "finder" app to use when playing hide-and-seek with a four-year-old:

https://emmy-finder.netlify.com/

### Get Started
1. Clone repo
2. Run `npm install`

### Dev server
`npm start`

### Build
`npm run build`
